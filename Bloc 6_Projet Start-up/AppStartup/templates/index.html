StartInvest - API

Welcome to the StartInvest - API.

Endpoints

  • /predict, method POST

    Predict the success of a Startup based on these features : ol>

  • Funding Total usd: the amount of the funding
  • Rounds: Raises capital through rounds
  • Creation date
  • First funding date
  • Last funding date

    • The expected input is a JSON with an input key followed by an array of inputs. Each inputs is an array of samples in the following order:

    1. Funding Total usd : an integer ex 10000
    2. Rounds : an Integer, ex 5
    3. Creation date (format : YYYY-MM-dd, ex 2001-01-11
    4. First funding date (format : YYYY-MM-dd, ex 2002-10-10
    5. Last funding date (format : YYYY-MM-dd, ex 2004-09-11

    JSON input example: 

                
        {
          "input": [[38730470,1,"2011-01-01","2011-01-01","2011-01-01"],[7998800,2,"2001-01-01","2008-07-10","2008-11-12"]]
        }

    Request example using curl: 

                			
				curl --globoff -i -H "Content-Type: application/json" -X POST -d "{\"input\": [[38730470,1,\"2011-01-01\",\"2011-01-01\",\"2011-01-01\"],[7998800,2,\"2001-01-01\",\"2008-07-10\",\"2008-11-12\"]]}" https://predictstartup.herokuapp.com/predict

    Expected output: 

                
        {
          "prediction": ['failure', 'success']
        }
